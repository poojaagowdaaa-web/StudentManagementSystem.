# Student Management System 🎓

A core Java console application built to streamline academic administration by managing student profile registration, tracking course enrollments, and structuring student records.

## 🚀 Key Features
* **Student Directory Management:** Capabilities to add, read, update, and remove active student profiles.
* **Course Enrollment Matrix:** Logic to dynamically assign students to specific core academic classes.
* **Object-Oriented Architecture:** Designed using clean Java class structures adhering to modular OOP principles.

## 🛠️ Technology Stack
* **Language:** Java SE (Standard Edition)
* **Concepts Used:** Encapsulation, Lists/Arrays, Data Modeling

## 🏃 How To Run Locally
1. Download or clone this repository to your machine.
2. Open the project folder in any Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Open `Main.java` and click **Run** to launch the interactive console interface.
